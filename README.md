# Claude Cowork Workshop — Data Kit

Sample data for the **Claude Cowork for Beginners** workshop (April 21, 2026 · AdVenture Media).

All files are fictional and built around one shared company — **Haven & Hearth Home Goods**, a mid-size direct-to-consumer + brick-and-mortar home goods brand. That keeps a coherent narrative running across the three hands-on tasks.

## How to use this kit

**Option A — Download everything (recommended):** Click the green **Code** button at the top of this page, then **Download ZIP**. Unzip it, and point Claude Cowork at the unzipped folder when prompted in the workshop.

**Option B — Clone in the terminal:**
```
git clone https://github.com/isaacjordan1022/cowork-workshop-data.git
```

## What's inside

### [Pack 1 — Messy folder](./pack-1-messy-folder) (file-organization task)
`messy-folder.zip` — ~35 files of realistic operator chaos. Vendor invoices, receipt photos, meeting notes, Slack screenshots, nested junk zips, a couple of stray spreadsheets. Unzip it, then tell Claude to organize the folder.

### [Pack 2 — Q1 Marketing P&L](./pack-2-q1-pnl) (spreadsheet analysis task)
Multi-tab workbook plus supporting files. A real-feeling Q1 marketing P&L with deliberate variances baked in. Tell Claude to analyze the variances and build a dashboard tab.

### [Pack 3 — Market research pack](./pack-3-research-pack) (document synthesis task)
Eight unstructured sources about the home-goods competitive landscape: analyst reports, a competitor landing-page scrape, two customer interview transcripts, a press release, internal analyst notes, and a 200-row customer survey. Tell Claude to synthesize them into a market brief.

## Prompts for each task

Each pack's folder contains a `README.md` with the exact workshop prompt and the shape of the expected output. Follow the prompt, then iterate.

## About the scenario

**Haven & Hearth Home Goods** is a fictional ~$10M/year DTC + retail home-goods brand (think bedding, bath, candles, kitchen tabletop). Four retail stores (NY, LA, Nashville, Austin). Online DTC, wholesale through Anthropologie and West Elm, Amazon presence, plus a subscription box called "Home Ritual." Everything in this kit — the numbers, the customer names, the competitor "RestWell" — is made up, but the shape and texture match what an operator in this business actually deals with day-to-day.
